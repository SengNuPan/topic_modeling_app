Welcome from document analysis with NMF!
  • The purpose of this project is to build an NLP web application that simplifies use of complicated NLP tools in an easy to use web interface.
  • For analysis of large document corpus, this app provide a simple mouse click with interactive visualization to help intuitive insights into your text data.
How is it built?
  • The web app is firstly deployed loacally and move to Heroku cloud platform to perform better and faster.
  • We can be consider a general tool for quickly discovering a set of topic and their distributions.
  • The app is capable of processing large documents due to NMF topic modeling algorithm alogn with Python Flask framework and NLTK modules!
